| Output Data                        | Description                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                       |                                                                                                                                                                                                                                                                  
|------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| AIWUM2-1_Monthly_1km_m3_2014_2021  | AIWUM 2.1 monthly (January-December) 1-km total water use rasters (2014-2021) in m3/year.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                         | 
| AIWUM2-1_Monthly_100m_m3_2014_2021 | AIWUM 2.1 monthly (January-December) 100-m total water use rasters (2014-2021) in m3/year. These rasters are summed up to the 1-km grid to create the 1-km water use rasters.                                                                                                                                                                                                                                                                                                                                                                                                                                                     | 
| AIWUM_Comparison                   | AIWUM 1.1 and 2.1 monthly growing season (April-October) 1-km total water use rasters (2014-2021) in m3/year. Also, contains the cumulative water use rasters for each year, wherein, the monthly rasters are summed up across the growing season for each year.                                                                                                                                                                                                                                                                                                                                                                  | 
| MAP_Extent_Rasters                 | These are the CDL 100 m rasters corrected for open-water bodies and cropped using the model extent.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                               |  
| Cleaned_MAP_GW_Data.csv            | Outlier-removed CSV file for model training, validation, and test. The column descriptions are as follows:<br/><br/>AF_Acre: monthly withdrawal in mm<br/>RO: monthly surface runoff in mm<br/>SM_IDAHO: monthly soil moisture in mm<br/>SSEBop: monthly actual evapotranspiration in mm<br/>Year: withdrawal report year<br/>crop: crop type (Corn, Cotton, Rice, Soybeans, Fish Culture)<br/>lat_dd: latitude of the field centroid in degrees<br/>long_dd: longitude of the field centroid in degrees<br/>ppt: PRISM 800-m monthly precipitation in mm<br/>tmax: PRISM 800-m monthly maximum air temperature in degree Celcius | 
| LGBM_Results.txt                   | LightGBM (LGBM) model results.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                    | 
| Models                             | All machine learning model analysis, objects, predictions, and results.                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                           |
