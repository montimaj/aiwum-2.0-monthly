| Input Data                        | Description                                                                                                                                                                                      | Source                                                                                                                                                                                                                                                                   |
|-----------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| Model Extent                      | Model extent shapefile and components                                                                                                                                                            | [Majumdar and others (2024a)](https://doi.org...)                                                                                                                                                                                                                        |
| Real-time                         | Contains daily withdrawal data from irrigation wells (2018-2021) along with the real-time flowmeter location shapefile.                                                                          | [Majumdar and others (2024a)](https://doi.org...)                                                                                                                                                                                                                        |
| AIWUM1-1_Monthly_total            | AIWUM 1.1 monthly 1-km total water use rasters (2014-2019) in m3/year.                                                                                                                           | [Bristow and Wilson (2023)](https://doi.org/10.5066/P9RGZOBZ); [Wilson (2021)](https://doi.org/10.3133/sir20215011)                                                                                                                                                      |
| CDL                               | USDA NASS CDL 30-m rasters (2014-2021) for the entire contiguous United States (CONUS).                                                                                                          | [Boryan and others (2011)](https://doi.org/10.1080/10106049.2011.562309)                                                                                                                                                                                                 |
| LANID                             | Annual Landsat-based Irrigation Dataset (LANID) 30-m Irrigation masks for the MAP.                                                                                                               | [Xie and others (2019)](https://doi.org/10.1016/j.isprsjprs.2019.07.005); [Xie and others (2021)](https://doi.org/10.5194/essd-13-5689-2021); [Martin and others (2023)](https://doi.org/10.5066/P9YWR0OJ); [Brandt and others (2021)](https://doi.org/10.5066/P9NAWU1U) |
| NHD                               | National Hydrography Dataset with partial modifiication to identify aquaculture in Mississippi Delta.                                                                                            | [Majumdar and others (2024b)](https://doi.org/10.5066/P9CET25K)                                                                                                                                                                                                          |
| Permitted_Boundaries              | Shapefiles components of permitted field boundaries within the Mississippi Delta. The 2021 data is replicated from the 2020 data.                                                                | [Majumdar and others (2024a)](https://doi.org...)                                                                                                                                                                                                                        |
| RO                                | TerraClimate product for monthly (January-December) 4-km surface runoff from 2014 to 2021.                                                                                                       | [Abatzoglou and others, 2018](https://doi.org/10.1038/sdata.2017.191)                                                                                                                                                                                                    |
| SM_IDAHO                          | TerraClimate product for the monthly (January-December) 4-km soil moisture data from 2014 to 2021.                                                                                               | [Abatzoglou and others (2018)](https://doi.org/10.1038/sdata.2017.191)                                                                                                                                                                                                   |
| SSEBop                            | Monthly (January-December) 1-km actual evapotranspiration from 2014 to 2021.                                                                                                                     | [Senay and others (2013)](https://doi.org/10.1111/jawr.12057)                                                                                                                                                                                                            |
| PPT                               | Monthly (January-December) PRISM 4-km precipitation from 2014 to 2021.                                                                                                                           | [Daly and others (2008)](http://doi.wiley.com/10.1002/joc.1688)                                                                                                                                                                                                          |
| TMAX                              | Monthly (January-December) PRISM 4-km maximum air temperature from 2014 to 2021.                                                                                                                 | [Daly and others (2008)](http://doi.wiley.com/10.1002/joc.1688)                                                                                                                                                                                                          |
| SWB                               | Contains daily 1-km irrigation demands (SWB_IRR) from 1999 to 2020 and 1-km hydrologic soil group (HSG). These datasets are not used as model predictors, but the user can optionally use these. | [Westenbroek and others (2021)](https://doi.org/10.3133/ofr20211008)                                                                                                                                                                                                     |
| VMP_Readings_Latest_2014_2020.csv | Voluntary Metering Program (VMP) annual irrigation well withdrawal data for wells in the Mississippi Delta. Data provided by Mississippi Department of Environmental Quality.                    | [Majumdar and others (2024b)](https://doi.org/10.5066/P9CET25K)                                                                                                                                                                                                          |
